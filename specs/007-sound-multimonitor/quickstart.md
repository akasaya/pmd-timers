# Quickstart: 通知音とマルチモニター 手動テストシナリオ

## シナリオ1: デフォルト通知音

1. アプリを起動（通知音は有効のまま）
2. 設定でタイマーを1分に短縮
3. タイマーをスタートして1分待つ
4. **期待**: セッション終了時に音が鳴る

## シナリオ2: 5秒タイムアウト

1. 5秒以上の長い音声ファイルを用意
2. 設定→通知音→「参照」でそのファイルを選択
3. 「プレビュー」ボタンを押す
4. **期待**: 5秒後に自動停止する
5. **期待**: 「⚠ 5秒でカットされます」の警告が表示されている

## シナリオ3: カスタム音声設定

1. 設定→通知音→「参照」で任意のWAVファイルを選択
2. 「プレビュー」で確認してOKを押す
3. タイマーを短縮して完了させる
4. **期待**: 設定したWAVファイルの音が鳴る

## シナリオ4: マルチモニター移動（2台以上の環境）

1. アプリを起動（メインディスプレイに表示）
2. ウィジェットをドラッグしてサブディスプレイに移動
3. **期待**: サブディスプレイに移動できる
4. アプリを再起動
5. **期待**: サブディスプレイの同じ位置に復元される
