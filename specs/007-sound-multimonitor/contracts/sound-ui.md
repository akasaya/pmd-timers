# UI Behavior Contracts: 通知音とマルチモニター

## 通知音

- `sound_enabled == True` かつセッション終了時 → 音声を再生する
- `sound_enabled == False` → 音声を再生しない
- 再生開始から5000ms後に音が止まっていない場合 → 強制停止
- `custom_sound_path` が有効なWAVファイルパス → そのファイルを使用
- `custom_sound_path` が空または無効 → `assets/sounds/notification.wav` を使用

## カスタム音声設定UI

- 「参照」ボタン押下 → *.wav フィルターのファイル選択ダイアログを開く
- ファイル選択後 → ファイル名をラベルに表示
- 「プレビュー」ボタン押下 → 選択ファイルを一時再生（5秒タイムアウト適用）
- 選択ファイルが5秒超 → 「⚠ 5秒でカットされます」を表示

## マルチモニター

- ドラッグ移動の範囲 → 全接続ディスプレイの仮想矩形（united geometry）
- ウィジェットが仮想矩形外になる場合 → 最寄りの端にクランプ
- 再起動時の位置復元 → 保存座標が全ディスプレイ外なら primaryScreen の左上にリセット
