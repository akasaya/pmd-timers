# Feature Specification: UIの多言語対応（i18n）

**Feature Branch**: `012-i18n-support`
**Created**: 2026-03-16
**Status**: Draft
**Input**: User description: "多言語に対応して"

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 英語UIに切り替えて使う (Priority: P1)

英語話者または日本語以外の環境のユーザーが、設定画面で言語を「English」に切り替えると、タイマーウィジェット・設定ダイアログ・トレイメニューのすべての表示が英語になる。

**Why this priority**: 現在UIは日本語のみ。英語対応が最も広いユーザー層へのリーチ拡大につながり、他言語追加の基盤にもなる。

**Independent Test**: 設定で English に変更して再起動し、すべての操作ラベルが英語で表示されることを確認できる。

**Acceptance Scenarios**:

1. **Given** アプリが日本語表示で起動している、**When** 設定ダイアログで言語を「English」に変更して再起動する、**Then** ウィジェット・設定ダイアログ・トレイメニューがすべて英語で表示される
2. **Given** 言語が「English」の状態で、**When** 設定ダイアログを開く、**Then** ダイアログ自身も英語で表示される
3. **Given** 言語を変更して保存した、**When** アプリを再起動する、**Then** 前回選択した言語が保持されている
4. **Given** 言語が「English」の状態で、**When** デスクトップ通知が発火する、**Then** 通知テキストも英語で表示される

---

### User Story 2 - 日本語に戻す (Priority: P2)

一度英語に切り替えたユーザーが、設定画面で再び「日本語」を選んで元に戻せる。

**Why this priority**: 切り替えができても戻せなければユーザーが困る。P1 と対になる基本動作。

**Independent Test**: 英語表示の状態から日本語に戻して再起動し、全テキストが日本語に戻ることを確認できる。

**Acceptance Scenarios**:

1. **Given** 言語が「English」に設定されている、**When** 設定ダイアログで「日本語」に変更して再起動する、**Then** 全UIが日本語表示に戻る

---

### Edge Cases

- 設定ファイルに未知の言語コードが保存されていた場合、日本語（デフォルト）にフォールバックする
- 翻訳が未定義のキーは、キー名をそのまま表示する（空欄にしない）
- 言語変更後、再起動するまでは変更前の言語が表示され続ける（即時切り替えは対象外）

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: 設定画面に言語選択項目を追加しなければならない（対応言語: 日本語・English）
- **FR-002**: 選択した言語は設定ファイルに保存され、再起動後も保持されなければならない
- **FR-003**: ウィジェットのフェーズ名・ボタン・右クリックメニュー・トレイメニュー・設定ダイアログ・デスクトップ通知のすべてのUI文字列が選択言語で表示されなければならない
- **FR-004**: 言語変更はアプリ再起動後に反映されなければならない（即時切り替えは対象外）
- **FR-005**: 翻訳が存在しないキーは日本語原文にフォールバックしなければならない
- **FR-006**: 将来の言語追加が容易な構造でなければならない

### Key Entities

- **言語設定**: 選択された言語コード（`ja` / `en`）を保持する設定項目
- **翻訳リソース**: 言語ごとのUI文字列マッピング（1言語1ファイル）

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 言語を English に切り替えて再起動したとき、ユーザーが目にするすべてのUI文字列が英語で表示される
- **SC-002**: 言語の切り替え操作（設定変更 → 保存 → 再起動）が1分以内に完了できる
- **SC-003**: 再起動後も選択した言語が維持されている
- **SC-004**: 日本語・英語いずれの言語でも、すべての既存機能が問題なく使用できる

## Assumptions

- 初期対応言語は「日本語」と「English」の2言語のみ
- 言語変更はアプリ再起動後に反映される（即時反映は将来課題）
- 翻訳対象はUI文字列のみ。設定ファイルのキー名・ログ出力は対象外
- 言語選択UIは設定ダイアログの「一般設定」グループに追加する
