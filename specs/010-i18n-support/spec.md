# Feature Specification: UIの多言語対応（i18n）

**Feature Branch**: `010-i18n-support`
**Created**: 2026-03-16
**Status**: Draft
**Input**: User description: "多言語に対応できると面白いのかな"

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 英語UIに切り替えて使う (Priority: P1)

英語話者または日本語以外の環境のユーザーが、設定画面で言語を「English」に切り替えると、タイマーウィジェット・設定ダイアログ・トレイメニューのすべての表示が英語になる。

**Why this priority**: 現在アプリのUIは日本語のみであり、英語対応が最も広いユーザー層へのリーチ拡大につながる。他言語はすべてこの仕組みの上に追加できる。

**Independent Test**: 設定を English に変更して再起動し、すべての操作ラベルが英語で表示されることを確認できる。

**Acceptance Scenarios**:

1. **Given** アプリが日本語表示で起動している、**When** 設定ダイアログで言語を「English」に変更して保存・再起動する、**Then** ウィジェット（フェーズ名・ボタンラベル）・設定ダイアログ・トレイメニューがすべて英語で表示される
2. **Given** 言語が「English」に設定されている、**When** 設定ダイアログを開く、**Then** 設定ダイアログ自身も英語で表示される
3. **Given** 言語を変更して保存した、**When** アプリを再起動する、**Then** 前回選択した言語が保持されている
4. **Given** 言語を「English」に変更した、**When** デスクトップ通知が発火する、**Then** 通知テキストも英語で表示される

---

### User Story 2 - 日本語に戻す (Priority: P2)

一度英語に切り替えたユーザーが、設定画面で再び「日本語」を選んで元に戻せる。

**Why this priority**: 切り替えができても元に戻せなければユーザーが困る。P1と対になる基本動作。

**Independent Test**: 英語表示の状態から日本語に戻して再起動し、全テキストが日本語に戻ることを確認できる。

**Acceptance Scenarios**:

1. **Given** 言語が「English」に設定されている、**When** 設定ダイアログで「日本語」に変更して再起動する、**Then** 全UIが日本語表示に戻る

---

### Edge Cases

- 設定ファイルに未知の言語コードが保存されていた場合、日本語（デフォルト）にフォールバックする
- 将来の言語追加時に翻訳が未完の文字列は、日本語原文をそのまま表示する（空欄にしない）
- 言語変更後、再起動するまでは変更前の言語が表示され続ける（即時切り替えは対象外）

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: 設定画面に言語選択項目を追加しなければならない（初期対応: 日本語・English）
- **FR-002**: 選択した言語は設定ファイルに保存され、再起動後も保持されなければならない
- **FR-003**: 対応するすべてのUI文字列（ウィジェットのフェーズ名・ボタン・トレイメニュー・設定ダイアログ・デスクトップ通知テキスト）が選択言語で表示されなければならない
- **FR-004**: 言語変更はアプリ再起動後に反映されなければならない（再起動なしの即時切り替えは対象外）
- **FR-005**: 翻訳が存在しない文字列は日本語原文にフォールバックしなければならない
- **FR-006**: 新しい言語を将来追加できる拡張可能な構造を持たなければならない

### Key Entities

- **LanguageSettings**: 選択された言語コード（例: `ja`, `en`）を保持する設定グループ
- **翻訳リソース**: 言語ごとのUI文字列マッピング。1言語につき1ファイル（またはセクション）で管理

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 言語を English に切り替えて再起動したとき、ユーザーが目にするすべてのUI文字列が英語で表示される
- **SC-002**: 言語の切り替え操作（設定変更 → 保存 → 再起動）が1分以内に完了できる
- **SC-003**: 再起動後も選択した言語が維持されている
- **SC-004**: 日本語・英語いずれかの言語で、すべての機能が問題なく使用できる（言語変更により機能が壊れない）

## Assumptions

- 初期対応言語は「日本語」と「English」の2言語のみとする
- 言語変更はアプリ再起動後に反映される（即時反映は将来課題）
- 翻訳対象は画面上のUI文字列のみ。設定ファイルのキー名・ログ出力などは対象外
- 言語選択UIは設定ダイアログの「ウィジェット設定」または新設する「一般設定」グループに追加する
