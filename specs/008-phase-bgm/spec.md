# Feature Specification: フェーズ別BGM再生

**Feature Branch**: `008-phase-bgm`
**Created**: 2026-03-16
**Status**: Draft
**Input**: 作業中と休憩中でそれぞれ別のBGMをループ再生できるようにしたい

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 作業中BGMの自動ループ再生 (Priority: P1)

ユーザーが作業セッションを開始すると、設定した作業用BGMが自動的にループ再生される。セッション終了（または一時停止）時にBGMが止まる。

**Why this priority**: BGMの核心機能。これだけで「作業に集中できる音楽環境」という価値が独立して届く。

**Independent Test**: 作業用BGMファイルを設定してタイマーを開始し、ループ再生されることを確認。タイマー終了時に停止することを確認。

**Acceptance Scenarios**:

1. **Given** 作業用BGMが設定・有効化済み、**When** WORKINGフェーズが開始される、**Then** BGMが最初からループ再生される
2. **Given** BGMが再生中、**When** WORKINGフェーズが終了する、**Then** BGMがすぐに停止する
3. **Given** BGMが再生中、**When** タイマーが一時停止される、**Then** BGMも停止する
4. **Given** タイマーが一時停止中、**When** タイマーが再開される、**Then** BGMが再び最初から再生される
5. **Given** 作業用BGMが無効化されている、**When** WORKINGフェーズが開始される、**Then** BGMは再生されない

---

### User Story 2 - 休憩中BGMの自動ループ再生 (Priority: P2)

休憩フェーズ（短休憩・長休憩）に入ると、設定した休憩用BGMがループ再生される。作業用と休憩用は独立して設定・有効化できる。

**Why this priority**: 作業BGMと同じ仕組みだが、気分の切り替えに使えるため重要。P1実装後に自然に拡張できる。

**Independent Test**: 休憩用BGMファイルを設定し、タイマーを休憩フェーズに進めてループ再生を確認。

**Acceptance Scenarios**:

1. **Given** 休憩用BGMが設定・有効化済み、**When** SHORT_BREAK または LONG_BREAK フェーズが開始される、**Then** 休憩用BGMがループ再生される
2. **Given** 作業BGMが再生中、**When** 休憩フェーズに切り替わる、**Then** 作業BGMが停止し、休憩BGMが開始される
3. **Given** 休憩BGMが再生中、**When** 次の作業フェーズに切り替わる、**Then** 休憩BGMが停止し、作業BGMが開始される
4. **Given** 休憩用BGMが無効化、作業用BGMが有効、**When** 休憩フェーズになる、**Then** 音楽は流れない

---

### User Story 3 - BGM音量の調整 (Priority: P2)

作業用・休憩用それぞれのBGM音量を個別に調整できる。通知音とは独立した音量設定。

**Why this priority**: 音量が固定だと環境に合わせられず使いにくい。ファイル選択と同時に提供すべき。

**Independent Test**: 音量スライダーを最小にしてBGMを再生し、音が聞こえなくなることを確認。

**Acceptance Scenarios**:

1. **Given** 設定ダイアログを開く、**When** 作業用BGM音量スライダーを操作する、**Then** 0〜100%の範囲で音量を設定できる
2. **Given** 音量を変更して保存、**When** 次の作業フェーズが開始される、**Then** 設定した音量でBGMが再生される
3. **Given** 音量を0%に設定、**When** BGMが再生される、**Then** 音は聞こえない（ミュート状態）
4. **Given** 作業用と休憩用で異なる音量を設定、**When** フェーズが切り替わる、**Then** それぞれの設定音量で再生される

---

### User Story 4 - 設定ダイアログからのBGM管理 (Priority: P2)

設定ダイアログで作業用・休憩用それぞれのBGMファイルを選択し、有効/無効を切り替え、プレビュー再生できる。

**Why this priority**: ファイル選択UIがないとBGM機能を使い始められない。US1・US2と並行して必要。

**Independent Test**: 設定ダイアログを開き、作業用BGMのファイル選択→プレビュー→保存ができることを確認。

**Acceptance Scenarios**:

1. **Given** 設定ダイアログを開く、**When** 作業用BGMの「参照」ボタンをクリック、**Then** WAVファイル選択ダイアログが開き、選択したファイル名が表示される
2. **Given** BGMファイルが設定済み、**When** プレビューボタンをクリック、**Then** 数秒間BGMが再生されて停止する
3. **Given** 設定ダイアログで変更を保存、**When** 次のフェーズが開始される、**Then** 保存されたBGM設定が反映される
4. **Given** BGMが有効、**When** チェックボックスをオフにして保存、**Then** 以降のフェーズでBGMが再生されない
5. **Given** 「デフォルトに戻す」を押す、**Then** BGM設定がすべてリセットされる（ファイルなし・無効・音量デフォルト）

---

### Edge Cases

- BGMファイルが削除されていた場合は再生をスキップし、アプリはクラッシュしない
- IDLEフェーズ（待機中）ではBGMは再生しない
- 作業BGMと休憩BGMに同じファイルを設定することも可能
- アプリ起動時点でタイマーがIDLEのため、BGMは再生しない

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: 作業フェーズ（WORKING）の開始時に作業用BGMがループ再生されなければならない
- **FR-002**: 休憩フェーズ（SHORT_BREAK・LONG_BREAK）の開始時に休憩用BGMがループ再生されなければならない
- **FR-003**: フェーズ切り替え時に前のBGMを停止し、新しいBGMを開始しなければならない
- **FR-004**: タイマー一時停止時にBGMを停止し、再開時に再びBGMを開始しなければならない
- **FR-005**: 作業用BGMと休憩用BGMをそれぞれ独立して有効/無効に切り替えられなければならない
- **FR-006**: 設定ダイアログでWAVファイルを選択できなければならない（作業用・休憩用それぞれ）
- **FR-007**: 設定ダイアログでBGMのプレビュー再生（数秒）ができなければならない
- **FR-008**: BGM設定（ファイルパス・有効/無効）はアプリ再起動後も保持されなければならない
- **FR-009**: BGMファイルが見つからない場合、再生をスキップしアプリは正常動作しなければならない
- **FR-010**: IDLEフェーズ・PAUSEDフェーズではBGMを再生してはならない
- **FR-011**: 作業用BGMと休憩用BGMそれぞれの音量を0〜100%の範囲で独立して設定できなければならない
- **FR-012**: 音量設定はアプリ再起動後も保持されなければならない

### Key Entities

- **WorkBgmSettings**: 作業用BGMのファイルパス・有効フラグ・音量（0〜100）
- **BreakBgmSettings**: 休憩用BGMのファイルパス・有効フラグ・音量（0〜100）
- **BgmService**: フェーズに応じてBGMの再生・停止・ループを管理するコンポーネント

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 作業フェーズ開始から1秒以内にBGMが再生開始される
- **SC-002**: フェーズ切り替えから500ミリ秒以内に前のBGMが停止し、新しいBGMが開始される
- **SC-003**: プレビューボタンを押してから1秒以内に音が鳴り始め、5秒以内に自動停止する
- **SC-004**: BGMファイルが存在しない状態でフェーズを進めてもアプリがクラッシュしない（100%）
- **SC-005**: 設定を保存してアプリを再起動した後もBGM設定が正しく復元される

## Assumptions

- BGMはWAV形式のみ対応（既存の通知音と同じ制限）
- ループ再生はファイルの末尾から先頭に自動で戻る（無限ループ）
- 音量は作業用・休憩用それぞれ独立して設定（デフォルト50%）
- BGM再生は通知音とは独立して動作する（同時再生を妨げない）
- デフォルトはBGM無効（ファイル未設定）
